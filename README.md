# portal-home-archive

用于归档和发布多个独立 HTML 页面的仓库。所有页面均为静态 HTML 文件，不依赖后端服务，可直接通过浏览器访问。

## 项目简介

本仓库旨在收集、整理并长期保存一系列独立的 HTML 页面。这些页面可能用于展示信息、提供工具界面或作为某些功能的入口。仓库本身不绑定任何特定域名或网站，仅作为页面的集中存储与版本管理。

## 目录说明

```
portal-home-archive/
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
└── index.html      # 可选：页面索引或导航入口
```

- `pages/`：每个页面建议以独立文件夹或文件形式存放，命名清晰，便于识别。
- `assets/`：存放多个页面共用的样式、脚本或图片资源，避免重复。
- `index.html`：可根据需要提供一个简单的页面列表，方便浏览。

## 页面归档说明

- 所有页面均为纯 HTML 文件，兼容主流浏览器。
- 页面之间相互独立，无强制依赖关系。
- 归档内容不包含任何动态后端逻辑或数据库连接。
- 如有外部资源引用（如 CDN 样式库），请在页面内明确标注来源。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面，或对已有页面进行优化。
- 添加页面时请遵循以下原则：
  - 文件命名简洁，使用小写字母与连字符（如 `example-page.html`）。
  - 如有必要，在页面内添加注释说明用途。
  - 避免引入过多外部依赖，保持页面轻量。
- 本仓库不承诺对归档内容进行实时更新或功能维护，仅保证文件的可访问性。
